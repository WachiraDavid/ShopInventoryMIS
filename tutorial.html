enlighten shop inventory MIS

This page helps you understand how to use the inventory management system

This is how you can use the Shop Inventory MIS

1. Create your account

- For you to use the Inventory MIS, you must be registered to your system.
- Fill the registration form(like the one shown on your left) the click the SUBMIT button to register

2. Log In

- Fill the Log in form then click SUBMIT to log in.
- Non the features in the Shop section can be accessed without Loging in.
- This avoids access by unauthorized users.

3. My Shop

- This page gives logged in users access to all the management features of the system
- These features include: -

  • Ability to stock the shop.
  • Ability to make sales in the system.
  • Ability to view the remaining and sold stock(quantity and prices).
  • Ability to view, download and/or print a profit and loss statement.
  • Access to this elaborate user guide/tutorial.

4. View statement

- This feature allows user access to a profit and loss statement which they can: -

  • View in the system
  • Download
    • ,and
  • Print

5. Stock Shop

- This feature allows users to stock the shop.
- The user can either: -

  • Fill a form to sock the shop for the first time, or
  • Use a simplified form to update the quantity of already existing stock items

6. Make Sale

- Here, users can click the sell button to add items to a cart that helps them;

  • View all items in the sale
  • Calculate the total price of the items on sale
  • Edit the cart to remove unnecessary items from the cart if necessary

7. View Inventory/Stock

- This feature allows users to view their various items that are in stock together with their details which include: -

  • Item Name
  • Item Quantity
  • Buying price
  • Selling Price
